# 📘 Coupon Management System — README

## 📌 1. Project Overview

The Coupon Management System is a full-stack web application that allows administrators to create, store, and manage discount coupons, while users can apply the best suitable coupon based on their cart and profile details. The project demonstrates backend API development, SQL database integration, and a clean frontend built using HTML, CSS, and JavaScript. It includes complete coupon filtering, discount calculation, usage tracking, and a working UI.
